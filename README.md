Village API :- 
Village API is a scalable B2B platform that provides access to India's village, district, sub-district, and state data through secure REST APIs. The platform includes API key management

### Frontend

* React 18
* Tailwind CSS
* React Query

### Backend

* Node.js
* Express.js
* JWT Authentication
* bcrypt

### Database & Caching

* Prisma ORM
* Redis (Upstash Redis)

### API & Documentation

* REST API


### Security

* JWT
* API Key Authentication
* Rate Limiting
* Helmet.js
* CORS
